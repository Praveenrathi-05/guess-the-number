# Guess the Number

A Python game where the computer generates a secret number and the player attempts to guess it within a limited number of tries.

## Features

- Multiple number ranges
- Random number generation
- Feedback for guesses
- Game restart functionality

## Concepts Practiced

- Event-driven programming
- User input handling
- Global and local variables
- Program state management

## What I Learned

This project introduced me to event-driven programming and managing application state based on user interactions.
